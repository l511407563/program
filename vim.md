#### 常用操作习惯整理
```
  模式切换：
   1. Esc -> NORMAL模式
   2. i -> INSERT模式
   
  光标位置切换(NORMAL模式)：
   gg -> 跳到文件开始
   G或shift+g  -> 跳到文件末尾
   h  -> 向左移动
   j  -> 向下移动
   k  -> 向上移动
   l  -> 向右移动
   H或shift+h  -> 跳到屏幕最顶上(行的最左侧)
   M或shift+m  -> 跳到屏幕中间(行的最左侧)
   L或shift+l  -> 跳到屏幕最下方(行的最左侧)
   w  -> 跳到下一个单词的开始
   e  -> 跳到下一个单词的结束
   b  -> 跳到上一个单词的开始
   A或shift+a或End  -> 跳到行尾
   I或shift+i或Home -> 跳到行首
   ngg或者:n+回车  -> 跳到第n行
   
```
