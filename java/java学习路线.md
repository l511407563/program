##### java学习路线
```
    1 -> 2 -> 3 -> 4 依赖关系，不可跳跃

    1. java基础(语言和数据结构)
        重点：
            语法基础
            面向对象
            集合类
            并发多线程
            异常处理
            泛型、注解、枚举
            IO
            反射

    2. java与数据库(数据库相关)
        重点：
            E-R图
            关系型数据库
            SQL语言
            事务处理
            JDBC
            连接池、数据源
            MySQL
            MyBatis(主流) 或者 Hibernate

    3. java web技术(网络相关)
        重点：
            前端
            HTTP
            Servlet
            Data Access
            Web
            AOP
            Spring Core
            IOC/ApplicationContext
            SpringBoot

    4. 项目实战(软件工程)
        重点：
            理解需求
            架构和设计
            项目代码组织
            知识综合运用
            Testing、Junit、Mockit等自动化测试工具
            性能
            健壮性
            Git
            Maven/Gradie

    5. 大型架构 + 专项精进
```
